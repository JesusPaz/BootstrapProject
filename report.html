Experience Portfolio Report About Me



  • Dificultades encontradas
    Afortunadamente en el desarrollo del ejercicio no tuve tantas dificultades. Pero existieron tres factores que me afectaron: Primero, construir una pagina de este tamaño requiere una buena inversion de tiempo, por lo que tuve que comenzar con tiempo de anticipación. Segundo, no me gustó el resultado final de la pagina index.html, siento que le dedique bastante tiempo pero no la pude mejorar en estetica. Tercero, fue dificil encontrar imagenes que se llevaran bien con los colores de la pagina, es una tarea que requiere mucho tiempo para obtener el mejor resultado.
  • Componentes Utilizados
    Los componentes nuevos utilizados para el desarrollo de este proyecto fueron:

    Carousel
    Se utilizó en la pagina portfolio.html, para representar imagenes dando la ilusión de que es el portafolio.
    Media Object
    Se agregó a esta pagina para realizar una lista con imagenes.
    Card
    Se implementó en la pagina portfolio.html, se uso para dar un aire diferente en donde se puedan apreciar categorias e imagenes.
    Progress
    Se utilizó en la pagina index.html, ya que era necesario representar una barra de progreso de las habilidades.
    Collapse
    Se implementó en la pagina portfolio.html, ya que permite agregar contenido de forma dinamica sin utilizar tanto espacio.
  • Conclusiones
    Fue un ejercicio que me hizo mejorar mis habilidades con bootstrap, ya que permite afrontar la realidad de como seria el desarrollo con esta herramienta. Ademas, es muy interesante tener que pensar en la arquitectura de una web ya que nunca tuve que enfrentarme a ese reto en el pasado.
